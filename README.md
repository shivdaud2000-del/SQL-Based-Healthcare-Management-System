# 🏥 SQL-Based Healthcare Management System 

## 📌 Project Overview
The Healthcare Management System is a SQL-based database project designed to efficiently manage hospital operations and healthcare records. The system organizes and analyzes data related to patients, doctors, appointments, treatments, and billing using MySQL.

This project demonstrates the practical implementation of relational database concepts, SQL query optimization, and healthcare data analysis for real-world hospital management.

---

# 🎯 Project Objectives

✅ Design a structured healthcare database using MySQL  

✅ Manage patient, doctor, appointment, treatment, and billing records efficiently  

✅ Perform SQL queries for healthcare analysis and reporting  

✅ Generate meaningful insights for hospital management  

✅ Improve understanding of DBMS concepts and healthcare analytics  

---

# 🗂️ Database Structure

The database consists of 5 interconnected tables:

| Table Name | Description |
|---|---|
| Patients | Stores patient details and personal information |
| Doctors | Contains doctor information and specialization |
| Appointments | Records doctor-patient appointment details |
| Treatments | Maintains diagnosis and treatment information |
| Billing | Stores billing and payment records |
